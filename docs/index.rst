.. Functional Fusion index

Functional Fusion Framework
===========================
The function fusion framework is a data structure / code based adopted by the Diedrichsenlab to learn brain organisation across functional imaging datasets.
The current emphasis is on the cerebellum and neocortex, but the framework is general and can be applied to any brain region.
